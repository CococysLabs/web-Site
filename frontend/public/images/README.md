# Imágenes del Proyecto COCOCYS

## Estructura de Carpetas

### `/logos`
Coloca aquí los 2 logos de la organización:
- `logo-principal.png` o `logo-principal.svg` - Logo principal de COCOCYS (recomendado: 200x60px o similar)
- `logo-secundario.png` o `logo-secundario.svg` - Logo secundario o de partner (recomendado: 200x60px o similar)

**Formatos recomendados**: SVG (vectorial), PNG con fondo transparente
**Peso máximo**: 100KB por archivo

### `/featured`
Coloca aquí las 2 imágenes destacadas:
- `imagen-1.jpg` o `imagen-1.png` - Primera imagen destacada (recomendado: 800x600px)
- `imagen-2.jpg` o `imagen-2.png` - Segunda imagen destacada (recomendado: 800x600px)

**Formatos recomendados**: JPG para fotos, PNG para ilustraciones
**Peso máximo**: 500KB por archivo
**Aspect ratio recomendado**: 4:3 o 16:9

## Placeholders Actuales

Mientras no subas las imágenes, se mostrarán placeholders con:
- Logos: Gradiente de colores de la marca
- Imágenes: Patrones de fondo sutiles

## Cómo Reemplazar

1. Coloca tus archivos en las carpetas correspondientes
2. Asegúrate de usar los nombres exactos mencionados arriba
3. Haz commit y push de los cambios
4. El deploy automático actualizará las imágenes
